试听课程

乐高机器人启蒙

适合 4–7 岁

适合 7–11 岁

一节课做出一个"接苹果"小游戏，学会坐标、随机数和得分变量的概念。

🕐 40 分钟👥 1 对 3📅 每周六/日下午

Python 探案学院

适合 10–13 岁

用 Python 写一个"猜数字"游戏，从 print 到 input 到 while 循环，一气呵成。

🕐 50 分钟👥 1 对 3📅 每周六下午

适合 12–16 岁

用 Arduino 组装一辆巡线小车，写一段控制代码让它沿着黑线跑完全程。

🕐 60 分钟👥 1 对 2📅 每周日上午

常见问题

试听课和正式课有什么区别？

试听课就是正式课的第一节——一样的老师、一样的教室、一样的上课方式。唯一的区别是上完不用交钱。

孩子从来没接触过编程，能跟上吗？

所有课程都从零开始设计。试听课就是给零基础的孩子准备的，你放心。

需要自己带电脑吗？

不需要。我们提供所有设备和教具，孩子空手来就行。课后作品可以带走。

试听课之后报名有优惠吗？

试听当天报名享受 9 折。另外兄弟姐妹同报、老学员推荐都有额外折扣，具体咨询陈老师。
