机器人 + C++ 竞赛营

硬件 + 算法双线推进，面向信奥赛和机器人竞赛。这堂课的目的是让孩子真正拥有一项可以写进简历的技能。

为什么是 C++，不是继续 Python

Python 适合入门和日常工具开发，但如果你要参加信息学奥林匹克竞赛（NOI 系列），你必须用 C++。这是硬性规定。

从 Python 转到 C++，孩子面临三个台阶：

  • 语法门槛：C++ 需要显式声明变量类型（int, float, char），需要手动管理内存，需要理解指针和引用。对习惯了 Python "怎么写都行"的孩子来说，这是第一次真正面对一门"严谨"的语言。
  • 编译概念：Python 是解释执行的，写完就跑。C++ 需要先编译成可执行文件再运行。这个"中间一步"是很多孩子的认知盲区。
  • 算法思维：竞赛题的难点不是语法，是算法——动态规划、贪心策略、图论、搜索剪枝。孩子要从"写代码"升级到"想清楚再写"。

我们的应对策略是：双线并行。一条硬件线，一条算法线。硬件负责保持兴趣（机器人真的会跑！），算法负责打好竞赛基础。两线在"控制逻辑"上天然交汇——机器人的巡线算法本质上就是贪心策略和 PID 控制。

硬件线：Arduino 从入门到创造

每个孩子领一套 Arduino 开发板 + 传感器套件，从零开始：

  • 第一阶段（1-8 课）：点亮 LED、控制舵机、读取传感器数据。理解"代码控制物理世界"的基本模型。
  • 第二阶段（9-16 课）：组装小车底盘，加入超声波避障、红外巡线、蓝牙遥控。每节课解决一个具体工程问题。
  • 第三阶段（17-32 课）：自选项目方向——智能家居控制器、自动浇花系统、机械臂、3D 打印外壳。老杨老师提供一对一技术指导。

算法线：从暴力到优化

硬件课占一半课时（前 24 节为主），算法课贯穿全期：

  • 入门（1-8 课）：C++ 基础语法、输入输出、数组、字符串处理。用洛谷入门题和 Codeforces 800 分题练手。
  • 进阶（9-20 课）：STL（vector, map, set, queue）、复杂度分析、贪心、二分、基础动态规划。刷洛谷普及组题目。
  • 冲刺（21-48 课）：图论基础（DFS, BFS, 最短路径）、高级 DP（背包、区间、状压）、搜索优化（剪枝、A*）。刷 CSP-J/S 真题。

小鹿老师会为每个有意参加竞赛的学生制定个人刷题计划，并定期做模拟赛和复盘。我们不保证每个孩子都能拿奖——但保证每个孩子都能在算法思维上跨一个台阶。

去年的事

小航来的时候 13 岁，Scratch 基础不错但没碰过代码。第一节课写 LED 闪烁，他盯着灯看了五分钟，然后问了一句："老师，这个灯闪的节奏能用代码控制吗？"一年后，他拿了东莞市机器人竞赛初中组一等奖。他的参赛作品是一个能自动识别垃圾种类并分拣的装置——Arduino 控制舵机和传送带，摄像头用 Python 做图像识别。一个项目跨了两种语言、两个平台。

这门课对升学的实际帮助

在东莞，科技特长生是初升高的一条正式路径。几所重点高中每年都有机器人/信息学竞赛的特长生名额。我们不做"包过"的承诺——那不是教育——但我们提供完整的竞赛规划、报名指导和模拟训练，让孩子如果真的感兴趣，有一条清晰的路可以走。
